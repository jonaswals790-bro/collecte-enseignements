# 📚 Système de Collecte de Données Terrain

Ce dépôt contient l'application de centralisation, d'analyse statistique et de diagnostic sociopédagogique concernant la conduite des exercices pratiques en Histoire-Géographie et Éducation à la Citoyenneté dans l'Arrondissement de **Ngaoundéré 1er**.

## 🚀 Fonctionnalités
* Formulaire complet sécurisé d'ajout, modification et suppression des données terrain (CRUD).
* Génération et extraction d'un **Rapport Institutionnel** au format HTML comprenant les tableaux, graphiques et analyses critiques.
* Sauvegarde synchronisée et persistante au format CSV et injection automatique dans ce fichier lisez-moi.
